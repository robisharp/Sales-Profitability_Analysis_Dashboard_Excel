# Sales-Profitability_Analysis_Dashboard_Excel
Sales Performance Dashboard (2024–2025) built using Microsoft Excel to analyze 10,000 sales records, visualize trends, and generate business insights using Pivot Tables, Charts, and interactive dashboard elements.

Project Description

This project analyzes sales performance data for the years 2024–2025 using Microsoft Excel. The dataset contains 10,000 rows of sales transactions, and the goal of this project is to clean the data, perform analysis, and build an interactive dashboard to visualize business insights.

The dashboard helps users understand sales trends, regional performance, product profitability, and salesperson performance through visual charts and summarized metrics.
